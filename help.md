## Export/Import
#### New Document
+ File ➡ New ➡ [Name] ➡ create
#### Open Document
+ File ➡ Open ➡ [Choose File] ➡ open
#### Save Document
+ File ➡ Save ➡ [Choose destination and name] ➡ save
